# 脉脉说说

---

最近做了个多 agent 量化项目，四个 AI 分析师全部看多茅台，然后 PM 把报告读完，决定卖。

原因是仓位超了 30% 的集中度上限，DCF 估值也显示贵了 32%。公司没问题，但价格已经把未来几年的增长提前定价进去了。

那天刚好是三个月回测的净值最高点，之后确实开始跌了。

---

做下来觉得多 agent 真正的价值不在于收益，在于分歧。分析师说买，PM 说卖，每个人的逻辑都完整保留着。如果是一个 LLM 直接回答「该不该买茅台」，大概率会把所有信息混在一起给你一个「综合来看偏多」，看不出这里面有矛盾。

三个月跑下来 agent +2.32%，同期买入持有 +0.50%。
